Problem Statement and Motivation:

This project aims to build an automated deep-learning system that accurately detects diseases from leaf images and causes of this disease, provide remedies to cure diseases, prevention tips.
Crop diseases cause major yield loss, and many farmers lack quick access to experts. A simple AI-based detection system can help farmers identify diseases early using just an image.
